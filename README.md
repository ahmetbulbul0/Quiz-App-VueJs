# Vue-Quiz-App

**_Vue Quiz App_**

## Technologies

-   **Html**
-   **Css**
-   **JavaScript**
-   **Vue**

## Project Setup Process

### Install npm (for npm packages)

```sh
npm install
```

### to run project in development environment

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with

```sh
npm run lint
```
